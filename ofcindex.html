O HTML5 é a versão mais recente do HTML e suporta novas funcionalidades de linguagem de marcação, com multimídia, novas tas, elementos e novas APIs. O HTML5 também suporta vídeo e áudio. HTML não oferece suporte nativo e aúdio e vídeo.

CSS, é uma linguagem usada para descrever a apresentação de documentos HTML. Em termos simples, o CSS define como os elementos de uma página web devem ser exibidos, incluindo cores, fontes, layout e outros aspectos visuais. O CSS trabalha em conjunto com o HTML, que define a estrutura do conteúdo, enquanto o CSS adiciona estilo e apresentação.

O que é a propriedade Display em CSS


Ela define se o elemento se comporta como um bloco, um elemento em linha ou se é ocultado. A propriedade display é fundamental para o layout e a disposição dos elementos em uma página web.

Display Características
Block Em HTML, um "bloco" refere-se a um elemento que ocupa toda a largura disponível em seu contêiner e, geralmente, começa em uma nova linha, criando uma estrutura visual distinta.
Flex Em "flex html", "flex" é geralmente usado no contexto de CSS Flexbox, um módulo de layout para organizar elementos numa dimensão (linha ou coluna) com mais controlo do que os layouts tradicionais.
Inline-Block Em CSS, display: inline-block combina características dos layouts inline e block. Elementos com essa propriedade se comportam como elementos inline, permitindo que vários elementos fiquem na mesma linha. No entanto, ao contrário de inline, eles também aceitam a definição de width e height, assim como elementos block
